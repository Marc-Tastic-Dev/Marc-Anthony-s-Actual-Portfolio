# Marc-Anthony's Enhanced Portfolio

# 🖤 Terminal Soul Drip 2.0

> A hyper-fast, framework-free portfolio site built by a 12-year-old full-stack dev who lives in the command line.  
> Enhanced with modern web standards, accessibility features, and performance optimizations.  
> Inspired by **Gruvbox**, **rauno.me**, and that raw **terminal hacker energy**.

---

## 🎯 What is this?

Terminal Soul Drip is a next-generation minimalist portfolio layout made with **pure HTML**, **CSS**, and **JavaScript**.  
It's for devs who believe in **maximum control**, **performance-first** workflows, and the **power of the terminal**.

**NEW IN 2.0:** Modern accessibility features, enhanced performance, SEO optimization, and professional-grade code quality.

---

## ⚡️ Features

### Core Features
- 🧱 **No frameworks** — just HTML, CSS, and vanilla JS
- 🎨 Gruvbox + Rauno inspired custom theme (`gruvrauno`)
- 🧭 Responsive sidebar navigation with intelligent scrollspy
- 🖥️ Terminal-styled outputs for flexing your dev story
- ⚙️ Fully modular layout with project cards + tag system
- 📱 Mobile responsive (yes, even with fixed sidebar)

### NEW: Enhanced Features
- ♿ **Full accessibility** — ARIA labels, keyboard navigation, screen reader support
- 🚀 **Performance optimized** — lazy loading, debounced scroll, optimized animations
- 🔍 **SEO ready** — meta tags, structured data, Open Graph tags
- ⌨️ **Keyboard shortcuts** — Ctrl/Cmd + 1-5 for quick navigation
- 🎭 **Loading animations** — smooth page transitions and typewriter effects
- 📊 **Performance monitoring** — FPS tracking and memory usage (dev mode)
- 📋 **Copy email** — one-click email copying with visual feedback
- 🎯 **Focus management** — proper focus indicators and tab navigation
- 🌟 **Modern CSS** — CSS custom properties, grid, flexbox, and animations

## 🎨 Customization

### Theme Colors
Modify CSS custom properties in `terminal-soul-drip.css`:
```css

  --accent-yellow: #fabd2f;
  --accent-orange: #fe8019;
  /* Add your custom colors */
}
```
